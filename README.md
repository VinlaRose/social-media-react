## Run Locally

Clone the project by adding the following command in the terminal.
Make sure your have installed GIT and added GIT to path.

```bash
  git clone 
```

Go to the project directory

```bash
  cd social-media-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
